Autonomous Arduino Robot

A self-navigating robot built with Arduino Uno, featuring obstacle avoidance, remote control capabilities, and various sensors for environmental awareness. This project combines hardware engineering with intelligent software algorithms.

üß† Autonomous Navigation

The robot uses a combination of ultrasonic sensing and servo-controlled scanning to map its environment. When an obstacle is detected, it stops, scans left and right, then chooses the path with the most clearance.

Obstacle Avoidance: Uses a simple but effective algorithm that prioritizes forward movement, scans for alternatives when blocked, and implements a basic memory system to avoid getting stuck in corners.
