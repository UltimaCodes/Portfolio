About

KeyLayer

KeyLayer is a keyboard automation tool for people who looked at their keyboard and thought, “I could do less.” It turns your computer into a button-pressing machine that does all the boring parts for you—like typing, clicking, or having patience.

It’s not a key remapper. It’s not just hotkeys. It’s more like: “What if this random $12 keypad controlled your job, your browser, and possibly everything else?”

Core Features

  • Device Isolation: Pick any keyboard and make it invisible to Windows. It now answers only to you.
  • Visual Macro Editor: Flowchart-style node editor. Connect boxes. Watch things happen. No coding. Mostly. (This may break on some pcs)
  • Layer System: Multiple keymaps per device. Toggle them, hold them, switch contextually
  • Macro Actions: Keypresses, app launching, screenshots, mouse control, file opening, HTTP requests, logic branches, loop
  • Import/Export: JSON, XML, AutoHotkey, PowerShell (Might not work at all oops)/li>
  • Macro Recorder: Press record. Harass your keyboard. Mess up the buffer system that took me wayy too long to do :D

Technical Stack

KeyLayer uses HidSharp for USB/HID communication, Newtonsoft.Json for the only normal JSON parsing experience, and WindowsAPICodePack to talk to Windows without completely offending it.

Technologies: C#, .NET, WPF, HidSharp, Newtonsoft.Json, Windows API Code Pack

Automation That Actually Does Stuff

  • Device-level control over which keyboard does what
  • Multi-layer macros for maximum overengineering
  • Live macro testing so you can break things in real time
  • Undo/redo support for when you immediately regret your entire layout
  • Encrypted profiles because I know what kind of person you are

Example Use Cases

  • Gaming: Fire off three cooldowns and launch Spotify—all with one button
  • Productivity: Type “Hello, just following up…” without actually typing it ever again
  • Development: Press one key, open your IDE, run a script
  • QA/Enterprise: Automate form input until the form automates you

Configuration Extras

  • Themes: light, dark, and “meh” (may not work)
  • Scan intervals and macro buffers for people who like sliders
  • Debug logs that will either help or bully you
  • Backups that mostly work (they don't)

Troubleshooting (Real Advice)

  • Device not detected: Try plugging it in. Really plug it in. Mean it.
  • Isolation fails: Probably another app being territorial. Close stuff until it works.
  • Macro doesn’t trigger: Check bindings.
  • Lag: Adjust config. Pretend you understand it.

Roadmap (Aspirational)

  • Linux/macOS support, eventually, probably, maybe
