Gorilla Warfare

A multiplayer FPS where everyone plays as a gorilla. It’s technically functional and very low effort. Built in Unity over the course of a week, mostly to refresh my knowledge on Photon PUN networking.

Gameplay Summary

  • Multiplayer lobby system using Photon PUN
  • Host-controlled match starting
  • Custom room names and usernames
  • Basic FPS movement: WASD, Jump, Sprint, Shoot
  • Scoreboard tracking kills and deaths
  • Red bullet impact squares that vanish after 30 seconds, for realism
  • Gravity-enabled death via falling into the void
  • Permanent T-posing gorilla player models, for atmosphere

Technical Implementation

Created in Unity using Photon PUN for networked multiplayer. The lobby system supports room listing, joining, naming, and host-based game control. A somehwat goofy FPS controller manages player input and shooting. Enemies do not exist. Everyone is the same gorilla. There are no teams. No win condition. Only meaningless violence.

Technologies: Unity 2022.3, C#, Photon PUN 2, Unity Input System

Design Notes

The map consists of ramps and walls. There is a gun. It makes no sound, but you’ll know it fired because a red square will appear briefly where the bullet hit. The T-posing gorillas are not placeholders. They are the vision.
